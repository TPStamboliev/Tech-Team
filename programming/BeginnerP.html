C++ Basics Test

1. What is the output of `int x = 5; cout << x;`?

 a) 5 b) 10 c) Error

2. What is the purpose of `#include`?

 a) Define a function b) Include a header file c) Declare a variable

3. What is the data type of `float x = 3.14;`?

 a) int b) float c) double

4. What is the output of `int x = 5; x = x * 2; cout << x;`?

 a) 5 b) 10 c) 20

5. What is the purpose of `using namespace std;`?

 a) Define a function b) Include a header file c) Use the standard library

6. What is the data type of `char x = 'A';`?

 a) int b) char c) float

7. What is the output of `int x = 5; int y = x + 2; cout << y;`?

 a) 5 b) 7 c) 10

8. What is the purpose of `main()`?

 a) Define a variable b) Include a header file c) Start the program execution

9. What is the data type of `bool x = true;`?

 a) int b) bool c) char

10. What is the output of `int x = 5; cout << x + 2;`?

 a) 5 b) 7 c) 10

Answers

Q1 Answer a) 5

Q2 Answer b) Include a header file

Q3 Answer b) float

Q4 Answer b) 10

Q5 Answer c) Use the standard library

Q6 Answer b) char

Q7 Answer b) 7

Q8 Answer c) Start the program execution

Q9 Answer b) bool

Q10 Answer b) 7
