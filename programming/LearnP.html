What is C++?

C++ is a high-performance, compiled, and general-purpose programming language. It was developed by Bjarne Stroustrup as an extension of the C programming language.

Variables and Data Types

In C++, a variable is a named storage location that holds a value. C++ has several built-in data types, including int, float, char, and bool.

Operators and Control Structures

C++ has various operators for performing arithmetic, comparison, logical, and assignment operations. It also has control structures such as if-else statements, switch statements, and loops (for, while, do-while).

Functions

In C++, a function is a block of code that can be called multiple times from different parts of a program. Functions can take arguments and return values.

Arrays and Vectors

In C++, an array is a collection of elements of the same data type stored in contiguous memory locations. A vector is a dynamic array that can grow or shrink in size as elements are added or removed.

Array Declaration:

  • To declare an array, specify the data type, the array name, and the number of elements it should store.
  • Syntax: dataType arrayName[arraySize];
  • Example: int x[6]; declares an integer array named x with a size of 6.

Accessing Array Elements:

  • Each element in an array is associated with an index (starting from 0).
  • Access elements using the syntax: array[index].
  • For example, x[0] is the first element, and x[5] is the last element.
